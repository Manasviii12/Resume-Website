# Resume-Website

## Overview
A professional, modern resume website for Manasvi Bhandari built with HTML and CSS, featuring a responsive design with an enhanced user interface.

## Design Enhancements

### Color Scheme
- **Modern Blue Palette**: Dark navy (#0f172a, #1e293b) backgrounds with vibrant cyan (#06b6d4, #0ea5e9) accents
- Professional gradient effects throughout for visual depth

### Navigation Bar
- Sticky navigation with gradient background and cyan bottom border
- Smooth animated underline effect on hover
- Gradient text styling for the logo
- Responsive layout for mobile devices

### Hero Section
- Eye-catching gradient text for name and title
- Subtle animated background shapes for depth
- Styled contact information cards with left borders
- Improved typography and spacing

### Card Sections
- Gradient top borders on all content cards
- Smooth hover animations (lift effect with enhanced shadow)
- Better visual hierarchy with improved spacing
- Left accent bar on section headings

### Skills Section
- Cyan checkmarks (✓) for each skill item
- Two-column layout with hover effects
- Responsive single-column on mobile

### Footer
- Gradient styling matching the navigation bar
- Cyan top border for consistency
- Professional typography

### General Improvements
- Rounded corners (15px) for modern aesthetic
- Enhanced box shadows for depth
- Better letter spacing and font weights
- Fully responsive design for all screen sizes
- Smooth transitions and animations throughout
